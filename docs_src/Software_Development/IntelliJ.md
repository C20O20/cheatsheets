# IntelliJ

## Shortcuts

    Search Anywhere     Double Shift
    Got to file         Ctrl + Shift + N
    Recent files        Ctrl + E
    Code Completion     Ctrl + Space 
    Parameters          Ctrl + P
    Highlight usages in file            Ctrl + Shift + F7
    Declaration of the current method   Alt + Q
    Code Templates      Ctrl + J
